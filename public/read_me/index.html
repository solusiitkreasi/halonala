Sistem Manajemen POS dengan Inventaris, HRM, Akuntansi

 
  • oleh: KAYO XLV
  • email: support@kayo45.com

KAYO XLV - POS adalah perangkat lunak yang akan membantu Anda mengelola inventaris, akuntansi, dan SDM. Kami percaya bahwa perangkat lunak ini cocok untuk model bisnis grosir dan eceran dan produk yang ideal untuk Super Shop mana pun. Perangkat lunak yang ramah pengguna ini sepenuhnya responsif dan memiliki banyak fitur. Semoga perangkat lunak ini dapat membantu mengelola inventaris bisnis Anda.

PERSYARATAN SERVER

Semua produk kami dirancang pada framework PHP Laravel paling populer. Anda harus memiliki persyaratan minimum untuk menjalankan semua aplikasi kami. Harap pastikan bahwa Anda telah melengkapi persyaratan ini.

  • Server Pilihan - Apache/Nginx
  • PHP Versi 7.4
  • Ekstensi PHP OpenSSL 
  • Ekstensi PHP PDO 
  • Ekstensi Info File PHP
  • Ekstensi Mbstring PHP 
  • Ekstensi Tokenizer PHP 
  • Ekstensi PHP Arsip Zip 
  • Mod Tulis Ulang Diaktifkan

 

Perlu diketahui jika Anda mencoba menginstal aplikasi di server lain, katakanlah LiteSpeed ​​atau IIS, Anda mungkin mendapatkan hasil yang tidak diinginkan. Kami tidak menyarankan Anda untuk menggunakan server selain Apache atau Nginx. Juga kami tidak menyediakan dukungan untuk instalasi di server selain Apache.

INSTALL

Localhost

Jika Anda menginstal di localhost, harap diingat bahwa Anda memerlukan Internet untuk instalasi. Setelah instalasi, Anda dapat menggunakan KAYO XLV tanpa Internet. Kami menyarankan Anda menggunakan XAMPP untuk localhost. Anda dapat menggunakan solusi lain seperti WAMP, MAMP dll. Juga dan proses instalasi untuk semuanya sama.

Buat folder di dalam folder htdocs di dalam instalasi XAMPP Anda. Sekarang salin folder zip yang Anda unduh dari Codecanyon ke folder htdocs dan unzip di sana. Ubah namanya menjadi apa pun yang Anda inginkan. Katakanlah, Anda menamainya- 'folder-saya'. Sekarang mulai XAMPP Anda (atau apa pun yang Anda gunakan) dan buka browser Anda dan akses KAYO XLV dari browser Anda seperti- localhost/my-folder/.

Saat menginstal, Anda mungkin menghadapi 'max execution time error', karena mengimpor dtabase memerlukan waktu cukup lama. Untuk mengatasi masalah ini, tingkatkan nilai 'max_execution_time' dan 'memory_limit' di file php.ini Anda. atur memory_limit=512M dan max_execution_time=480. untuk referensi- Cara mengedit file php.ini di localhost (XAMPP)

Hosting Daring

Unggah folder zip yang Anda unduh dari Codecanyon ke hosting Anda dan unzip. Harap pastikan Anda mengonfigurasi pengaturan hosting web Anda, sehingga menampilkan file dan folder tersembunyi (untuk referensi- Cara mengaktifkan file dan folder tersembunyi di cpanel ) Ini untuk memastikan bahwa jika Anda menyalin/memindahkan konten dari folder yang telah dibuka zipnya ke folder mana pun lokasi lain, Anda menyalin semua file termasuk file '.htaccess', '.env' yang diperlukan agar perangkat lunak berfungsi dengan baik. Sekarang Anda dapat mengakses folder tempat Anda memiliki KAYO XLV dari browser Anda.

Sekarang ikuti proses instalasi di bawah ini.

Langkah 1

  • Harap baca perjanjian lisensi sebelum melanjutkan.
  • Anda harus menerima dan melanjutkan untuk pergi ke langkah berikutnya.

Langkah 2

  • Sistem akan secara otomatis memeriksa persyaratan server. Jika semua persyaratan terpenuhi, Anda dapat melanjutkan untuk tindakan selanjutnya.

Langkah 3

  • Silahkan masukan kode pembelian.
  • Masukkan host basis data Anda misalnya (localhost).
  • Nama pengguna basis data Anda misalnya (root).
  • Kata sandi basis data Anda (jika ada).
  • Nama database yang telah Anda buat.
  • Klik kirim.

Langkah 4

  • Selamat! Anda telah berhasil menginstal KAYO XLV - POS.
    Note : Proses otomatis ini seharusnya menghapus folder install yang ada di dalam folder project. Folder project anda berisi folder 'install', maka silahkan hapus folder tersebut.

Kesalahan Umum

Jika Anda menghadapi 500 server error setelah menginstal perangkat lunak, perbarui versi php Anda ke 7.4. Jika Anda masih mendapatkan error 500 setelah memperbarui versi php, buka file '.env' Anda dan ubah nilai 'APP_DEBUG' menjadi true. Anda akan menemukan file '.env' di folder root (KAYO XLV) Dan kemudian pergi ke halaman lagi di mana Anda mendapatkan 500 kesalahan server. Anda akan melihat deskripsi kesalahan aktual sekarang. Silakan ambil tangkapan layar dan kirimkan bersama dengan detail akses cpanel Anda, sehingga kami dapat memeriksanya.

Mesin Timbangan Berat

KAYO XLV - POS kompatibel dengan mesin timbangan. Tetapi Anda harus mengikuti aturan dengan ketat. Pada mesin timbangan berat badan mesin akan memberikan label berupa 13 digit barcode. Perhatikan gambar berikut:

Disini 7 digit pertama adalah kode produk yang harus sama dengan kode produk KAYO XLV - POS. 5 digit berikutnya berisi bobot dan digit terakhir adalah angka acak. Setiap kali Anda meletakkan produk pada mesin skala berat, itu akan menghasilkan label jenis ini secara otomatis. Maka yang harus Anda lakukan adalah memindai label ini. Info dan berat produk akan dihitung secara otomatis dengan KAYO XLV - POS. Namun kode produk harus sama baik untuk KAYO XLV - POS maupun mesin timbangan.

Cetak Barcode

Anda dapat mencetak barcode dengan KAYO XLV. Ukuran kertas yang tersedia adalah 36mm, 24mm dan 18mm. Apa pun selain itu mungkin tidak berfungsi dengan benar. Untuk mencetak barcode kami sangat menyarankan untuk menggunakan Brother Label Printer.

Jika produk tidak dapat memenuhi tujuan, pengguna akan diberi tahu di dasboard. Untuk membuat fitur ini terotomatisasi, Anda harus mengatur tugas cron dengan benar.

Pertama pergi ke cpanel dan pergi ke cron job settings

Kemudian atur tugas cron seperti yang dijelaskan di bawah ini. Di sini tugas cron diatur sekali sehari. Jadi sistem akan memeriksa pada pukul 12:00 jika ada produk yang tidak dapat memenuhi target penjualan harian untuk hari sebelumnya. Produk yang tidak dapat memenuhi target penjualan harian akan dimasukkan ke dalam laporan target penjualan harian.

Note: Jika Anda menghadapi masalah apa pun saat menyiapkan tugas cron, silakan hubungi di Support.

PENGATURAN

Tambah Role

Anda dapat membuat, mengedit, dan menghapus peran pengguna. Anda dapat mengontrol akses pengguna dengan mengubah izin peran. Jadi, di bawah peran tertentu pengguna memiliki akses khusus atas perangkat lunak ini

Tambah Gudang

Anda dapat membuat, mengedit, dan menghapus gudang. Anda juga dapat mengimpor gudang dengan file CSV. Anda harus mengikuti instruksi untuk mengimpor data dari CSV.

Tambah Grup Pelanggan

Anda dapat membuat, mengedit, dan menghapus grup pelanggan. Kelompok pelanggan yang berbeda memiliki harga yang berbeda atas produk tersebut. Anda dapat memodifikasi ini dengan mengubah persentase harga dalam modul Customer Group.

Anda juga dapat mengimpor grup pelanggan dengan file CSV. Anda harus mengikuti instruksi untuk mengimpor data dari CSV.

Tambah Brand

Anda dapat membuat, mengedit, dan menghapus merek produk. Anda juga dapat mengimpor merek dengan file CSV Anda harus mengikuti instruksi untuk mengimpor data dari CSV.

Tambah Satuan

Anda dapat membuat, mengedit, dan menghapus satuan produk. Anda juga dapat mengimpor merek dengan file CSV. Anda harus mengikuti instruksi untuk mengimpor data dari CSV.

Tambah Pajak

Anda dapat membuat, mengedit, dan menghapus pajak produk yang berbeda. Anda juga dapat mengimpor pajak dengan file CSV. Anda harus mengikuti instruksi untuk mengimpor data dari CSV.

Dan Anda dapat mencari, mengekspor, dan mencetak data dari tabel yang telah kita bahas di bagian Produk.

General Settings

Anda dapat mengubah Judul Situs, Logo Situs, Mata Uang, Zona Waktu, Akses Staf, Format Tanggal, dan Warna Tema dari pengaturan umum

Profil pengguna

Anda dapat memperbarui info profil pengguna dari modul ini

Pengaturan POS

Anda dapat mengatur pengaturan POS Anda sendiri dari modul ini. Anda dapat mengatur pelanggan default, biller, gudang dan berapa banyak produk Unggulan yang akan ditampilkan di modul POS. Anda harus mengatur kunci publik dan pribadi Stripe Anda untuk Pembayaran Kartu Kredit Untuk menerapkan pembayaran dengan Paypal Anda harus membeli api langsung dari Paypal. Anda juga perlu mengisi informasi berikut.

HRM Setting

Anda dapat mengatur waktu CheckIn dan CheckOut default di Pengaturan HRM.

SMS Setting

Anda dapat menggunakan layanan Bulk SMS melalui Twilio dan Clickatell. Anda hanya perlu mengisi informasi dengan benar untuk mengaktifkan layanan ini. Harap berikan kode negara untuk mengirim sms.
